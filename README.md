# Comissão Agendador

App de controle de comissão escalonada com banco de dados na nuvem.

## Como subir no Glitch

1. Acesse **glitch.com** e crie uma conta gratuita
2. Clique em **"New Project"** → **"Import from GitHub"**  
   (ou use "glitch.new" e depois importe o ZIP)
3. Faça upload dos arquivos:
   - `server.js`
   - `package.json`
   - `public/index.html`
4. O Glitch instala as dependências automaticamente
5. Seu app estará disponível em: `https://NOME-DO-PROJETO.glitch.me`

## Como compartilhar com o agendador

Basta enviar o link `https://NOME-DO-PROJETO.glitch.me` por WhatsApp.  
Os dois acessam o mesmo banco de dados em tempo real.

## Modelo de comissão

- **Fase 1** (1 a 15 agendamentos): 5%
- **Fase 2** (16 a 25 agendamentos): 7%  
- **Fase 3** (26 a 30 agendamentos): 10%
